# ReadingExpericence

串接後端Restful API書籍資料，以MVC方式實作一個讀書心得App，並嘗試實作側邊欄。  
有登入、個人資料、瀏覽全部書籍、瀏覽單一書籍、評分留言、編輯留言、刪除留言...等功能。

Demo影片：https://www.youtube.com/watch?v=1j-q4P26k4Y
